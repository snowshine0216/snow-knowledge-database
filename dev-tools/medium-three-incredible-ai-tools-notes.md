## Key Takeaways
- **Google AI Studio App Builder** — free vibe-coding platform at aistudio.google.com; builds functional internal or public apps in minutes from a brief text description
- Wordle clone built in two prompts; pixel-art color converter for a clothing brand built in a few prompts — both worked end-to-end without manual code edits
- Apps can be deployed directly from AI Studio for public use
- **Replicate** (replicate.com/explore) — aggregates thousands of generative AI models in one platform: image generation (Seedream 4.5), language models (DeepSeek), upscaling, video generation, audio transcription, OCR, shape detection, sound effects
- Replicate's composable pipeline: generate prompt → generate image → upscale → animate into video, all within one platform
- **Rhyme.ink** — voice-first writing canvas; dictate text, apply markdown formatting (headings, bullets, bold), and run AI commands entirely by voice ("Hey Rhyme, convert to prose and fix grammar")
- Rhyme features: custom word shortcuts (single word → full paragraph), full markdown, cursor navigation by voice, unlimited undo, focus mode, free tier

## Insights
- Google AI Studio Builder differentiates from other vibe-coding tools by producing *deployable* apps with minimal prompting — worth evaluating against Lovable/Bolt for internal tooling
- Replicate is a practical way to access SOTA models without managing infrastructure; useful when you need to chain models (e.g., LLM → image → upscale) without building a pipeline from scratch
- Rhyme.ink targets the dictation-to-formatted-document workflow — relevant for async write-ups, engineering docs, or postmortems where speed of capture matters more than keyboard input
- All three tools have free tiers, lowering the barrier to experimentation
