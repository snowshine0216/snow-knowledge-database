# Interview Summaries

Store all interview video summaries here.

## Naming convention
- `<meaningful-title-slug>_<video-id>.md`
- Example: `andrej-karpathy-on-code-agents-autoresearch-and-the-loopy-era-of-ai_kwSVtQ7dziU.md`

## Recommended flow
1. Run the yt-video-summarizer skill on one video URL.
2. Save the output into a new markdown file in this folder.
3. Keep the 5-section structure for consistency:
   - Video Info
   - Key Points
   - Timeline
   - Takeaways
   - Source Notes
