# OpenClaw Skills：AI 的岗位培训包

Skills 是封装好的 AI 能力包，解决"每次重新教、容易忘、上下文负担重"三大痛点。安装方式极简——直接和龙虾对话说"帮我装 X Skill"即可。装上即拥有新能力，永久可用。

## Key Concepts
- **Skills 本质**：封装好的岗位培训包，AI 装上即会，无需每次重复配置
- **三大痛点**：无 Skills 时每次重复劳动 + 忘记偏好 + 上下文负担，Skills 全部解决
- **对话即安装**：直接告诉龙虾"帮我安装 X"，AI 自主执行安装流程
- **Agent Rich 示例**：网页/社交媒体内容爬取 Skill，装完即可让龙虾搜索 Twitter/微博

## Key Takeaways
- Skills = AI 能力的持久化单元，类比给员工的岗位手册，装一次永久有效
- 安装过程完全通过对话完成，不需要技术背景
- Skill 让龙虾从"聊天工具"变成"真正会干活的数字员工"
