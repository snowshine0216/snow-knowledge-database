# AI Tools Roundup

A reference overview of three AI tools with free tiers, covering vibe-coding, model aggregation, and voice-first writing.

## Google AI Studio App Builder

- **URL:** https://aistudio.google.com/
- **Category:** Vibe-coding / rapid app prototyping
- **What it does:** Generates functional web apps from brief text descriptions, deployable directly from the platform
- **Differentiator:** Produces deployable apps with minimal prompting -- fewer iterations than many competitors
- **Use case:** Internal tools, quick prototypes, simple public-facing apps
- **Example:** A working Wordle clone built in two prompts; a pixel-art color converter built in a few prompts, both without manual code edits

Worth evaluating against tools like Lovable and Bolt for internal tooling use cases.

## Replicate

- **URL:** https://replicate.com/explore
- **Category:** Model aggregation platform
- **What it does:** Provides access to thousands of generative AI models in one place -- image generation, language models, upscaling, video generation, audio transcription, OCR, and more
- **Key capability:** Composable pipelines -- chain models together (e.g., generate prompt, generate image, upscale, animate to video) within one platform
- **Notable models:** Seedream 4.5 (image), DeepSeek (language)
- **Use case:** Accessing state-of-the-art models without managing infrastructure; useful when chaining multiple model types in a single workflow

## Rhyme.ink

- **URL:** https://rhyme.ink
- **Category:** Voice-first writing tool
- **What it does:** Dictation-to-formatted-document canvas with voice-controlled Markdown formatting and AI commands
- **Features:**
  - Apply headings, bullets, bold via voice
  - AI commands by voice (e.g., "convert to prose and fix grammar")
  - Custom word shortcuts (single word expands to full paragraph)
  - Cursor navigation by voice
  - Unlimited undo, focus mode
- **Use case:** Async write-ups, engineering docs, or postmortems where speed of capture matters more than keyboard input

## Caveats

These are personal recommendations (not sponsored), not benchmark comparisons. No performance data, API access details, rate limits, or enterprise pricing are covered. All three offer free tiers suitable for experimentation.
