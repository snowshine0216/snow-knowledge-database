# OpenClaw 办公 Skills Top 10

从 ClawHub 的 1.3 万个 Skills 中精选出适合日常办公的 10 个高频通用技能包，目标是快速搭建"龙虾办公室"基础配置，而不是在海量 Skills 中迷路。

## Key Concepts
- **精选标准**：高频、通用、安全——三条件缺一不可，排除低质/行业专用/低可信 Skills
- **核心 10 类**：Google Search、Summarizer、Git/GitHub、Translation、Web Research、File Manager、Video Summarizer、Data Analysis、Email Assistant、Calendar/Task
- **组合威力**：单 Skill 价值有限，Web Search + Summarizer 组合可实现 AI 行业晨报自动化，2 分钟取代 1 小时人工
- **安装起点**：优先装前 3 个（Google Search + Summarizer + Git）形成最小可用闭环

## Key Takeaways
- 先装 3 个核心 Skill，再按实际需求扩展，避免"什么都装但什么都不用"
- Skill 组合是解锁 Agent 工作流的关键——单点工具 < 组合自动化流程
- 文件管理 Skill 对内容创作者尤其实用，解决素材"爆炸"痛点
