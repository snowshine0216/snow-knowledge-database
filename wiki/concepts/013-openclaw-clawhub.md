# ClawHub：OpenClaw 官方技能市场

ClawHub（clawhub.ai）是 OpenClaw 生态的官方 Skills 应用市场，目前收录超过 1.3 万个技能包。数量庞大但质量参差——约 20% 是真正可用的高质量 Skills，其余多为重复发布或低价值内容。有效使用 ClawHub 的关键在于掌握正确的搜索语言。

## Key Concepts
- **ClawHub 定位**：OpenClaw 官方 Skills 市场，相当于 AI Agent 生态的"应用商店"，一站式获取、安装 Skills
- **质量分布**：1.3 万+ Skills 中约 20% 为优质可用，大量重复/垃圾/恶意内容混杂其中
- **搜索语言转换**：市场基于开发者技术标签，需将业务需求翻译为技术术语（"配图" → `image generation`，"抓取" → `scraper`）
- **质量信号**：Stars 数（社区认可）+ Downloads 数（实际使用量）是快速筛选的最可靠指标
- **安装入口**：直接在 OpenClaw 对话中说"帮我安装 [Skill名称]"，无需手动配置

## Key Takeaways
- 不能无脑安装 ClawHub 上的 Skills，需主动筛选
- 技术英文关键词比中文业务描述搜索效果更好
- Stars + Downloads 双高的 Skill 优先考虑，两者都低的谨慎安装
- 5 分钟内用正确关键词可找到并安装符合业务需求的 Skill
