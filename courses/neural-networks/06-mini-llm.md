# Large Language Models explained briefly

## Metadata
- Topic page: https://www.3blue1brown.com/topics/neural-networks
- Lesson page: https://www.3blue1brown.com/lessons/mini-llm
- Video: https://www.youtube.com/watch?v=LPZh9BOjkQs
- Date: 2024-11-20

## Summary
- Defines an LLM as a function that predicts a probability distribution over the next token.
- Explains chatbot generation as repeated predict-and-sample steps conditioned on all prior text.
- Emphasizes that training means tuning a huge number of parameters on massive text corpora.
- Clarifies why randomness in sampling improves naturalness compared with always taking only the top-probability token.
- Serves as a conceptual bridge before the deeper transformer internals in subsequent chapters.

## Key Ideas
- next-token prediction
- probability distributions
- sampling
- parameter training
- LLM generation loop
