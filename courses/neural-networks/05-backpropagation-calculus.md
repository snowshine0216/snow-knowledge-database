## Summary
- Turns the previous intuitive story into explicit calculus and matrix equations.
- Derives partial derivatives of cost with respect to biases and weights via intermediate variables (pre-activations and activations).
- Introduces the recursive delta formulation that transports output error backward layer by layer.
- Shows compact vectorized formulas that map directly to efficient software implementations.
- Generalizes from one sample to full-dataset training by averaging gradient contributions.

## Key Ideas
- partial derivatives
- delta recursion
- vectorization
- matrix-form backprop
- dataset-level gradient
