# Gradient descent, how neural networks learn

## Summary
- Defines a cost function that measures how wrong the network is across labeled training examples.
- Treats all parameters as coordinates in a high-dimensional landscape where lower cost means better performance.
- Explains the gradient as the direction of steepest increase, so moving in the negative gradient direction reduces cost fastest locally.
- Introduces learning rate as the step-size tradeoff between stability and speed.
- Connects theory to practice with mini-batch/stochastic updates, which make training computationally feasible.

## Key Ideas
- cost function
- gradient vector
- negative gradient update
- learning rate
- stochastic gradient descent
