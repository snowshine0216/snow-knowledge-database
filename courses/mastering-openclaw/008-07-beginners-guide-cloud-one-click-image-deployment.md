## 云端部署的场景与优势

云端部署适合以下情况：
- 开发者、运行代码需要 24 小时不间断服务
- 不想让个人电脑持续开机
- 需要从任何设备访问 AI

**核心优势**：
- 24/7 持续运行，不依赖个人电脑
- 运维全托管（水电装修都搞定），零门槛上手
- 可让完全不懂服务器的小白轻松拥有自己的 AI

---

## 主流云平台对比

| 云平台 | 特点 | 适用人群 |
|--------|------|---------|
| **阿里云** | 资源最丰富，镜像一键安装、开箱即用，社区文档齐全 | 喜欢自研、需要更多功能扩展 |
| **腾讯云** | 轻量应用服务器体验佳，镜像部署简单 | 快速上手，性价比优先 |
| **扣子（Coze）** | 无服务器方式，可直接在平台部署 | 零运维需求 |

其他国内云厂商（华为云、百度云等）同样支持一键镜像，大同小异。

## 腾讯云一键镜像部署实战

### 步骤

**1. 搜索轻量应用服务器**
- 进入腾讯云控制台
- 搜索"宝塔"或 OpenClaw 相关关键词
- 找到"轻量应用服务器"中的镜像市场

**2. 选择 OpenClaw 镜像**
- 在镜像列表中找到 OpenClaw 一键部署镜像
- 查看配置说明和费用

**3. 购买配置**
- 基础套餐即可（满足日常使用）
- **地区选择**：建议新加坡（国际访问更顺畅）
- 按年购买更划算

**4. 启动实例**
- 购买后进入实例管理
- 系统会自动完成 OpenClaw 安装
- 等待初始化完成

**5. 连接渠道**
- 配置飞书/微信/钉钉 Channel（同本地安装方式）
- 安装需要的 Skills

## 部署后配置

云端部署完成后与本地部署配置流程一致：
1. 选择 AI 模型（填入 API Key）
2. 连接通讯渠道（飞书/微信）
3. 安装所需 Skills

由于云端 24h 运行，可以给 OpenClaw 更多的主动任务（如定时提醒、自动汇报等）。

---

## 成本与注意事项

**成本构成**：
- 服务器费用：轻量服务器按年付，百元级别
- **LLM API 费用（主要成本）**：每月可能十几元到几十元甚至更高，取决于使用频率

> **警示**：若 Skills 配置不当或 AI 进入循环，API 消耗可能激增（几十到上百元）。务必关注使用量。

**国内用户推荐方案**：
- 一键镜像部署 + 多平台 Channel 接入
- 控制 API 调用成本，每月控制在百元以内完全可行

**结论**：云端一键部署真正实现了"小白也能行"——不懂代码的人三十分钟内完成部署，7×24h AI 随时待命，这才是云端部署的核心价值。
