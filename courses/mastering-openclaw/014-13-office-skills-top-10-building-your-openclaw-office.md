## 为什么要精选办公 Skills

ClawHub 上虽有 1.3 万个 Skills，但对于办公场景来说，大多数是不适用的。本讲聚焦：

- **高频使用**：几乎每天都用得到
- **通用场景**：不限于特定行业，大多数人都适用
- **安全可信**：官方认证或社区高评分，避免恶意/低质 Skill

> 目标：帮你快速装好"龙虾办公室"的基础配置，而不是让你在 1.3 万个 Skills 里迷路。

## Top 10 办公 Skills 精选

### 1. Google Search / Maps
- **用途**：联网搜索、地图查询、POI 信息获取
- **适用人群**：几乎所有用户
- **搜索词**：`google search`、`google maps`

### 2. 内容总结（Summarizer）
- **用途**：网页、视频、文章一键总结
- **典型场景**：快速消化长文、会议纪要、新闻
- **搜索词**：`summarizer`（选 stars 最高版本）

### 3. 版本管理（GitHub/Git 集成）
- **用途**：文件版本控制，回溯修改历史
- **非技术场景**：内容创作的稿件版本管理（v1/v2/v3 → 找回旧版本）
- **搜索词**：`git`、`github`
- **国内替代**：OpenCode、Gitee 相关 Skills

### 4. 翻译（Translation）
- **用途**：中英互译、多语言内容处理
- **特点**：结合龙虾的上下文理解，比单纯翻译 API 更自然
- **搜索词**：`translation`、`translator`

### 5. 网页搜索聚合（Web Research）
- **用途**：多源信息搜集，自动汇总搜索结果
- **搜索词**：`web search`、`research`

### 6. 文件管理（File Manager）
- **用途**：批量整理、重命名、分类本地文件
- **痛点场景**：内容创作者素材管理混乱问题
- **搜索词**：`file manager`、`file organizer`

### 7. 视频总结（Video Summarizer）
- **用途**：YouTube/B 站视频内容提取摘要
- **搜索词**：`video summarizer`

### 8. 数据分析（Data Analysis）
- **用途**：Excel/CSV 数据处理、图表生成
- **搜索词**：`data analysis`、`spreadsheet`

### 9. 邮件助手（Email Assistant）
- **用途**：起草邮件、回复建议、邮件分类整理
- **搜索词**：`email`、`email assistant`

### 10. 日历/任务管理（Calendar/Task）
- **用途**：日程安排、提醒设置、任务追踪
- **搜索词**：`calendar`、`task manager`

## 组合 Skill 实战案例

### 案例 1：AI 行业资讯晨报自动化

**痛点**：每天手动搜索 → 打开网页 → 阅读 → 总结，耗时 1 小时以上

**解决方案**：Web Search Skill + Summarizer Skill 组合

```
用户：帮我找 2026 年 AI Agent 市场的最新动态，整理成晨报格式
龙虾：[Web Search] 搜索相关关键词 → [Summarizer] 提炼要点 → 输出结构化晨报
```

**效果**：
- 原来：人工搜索、拼接、总结 → 30-60 分钟
- 现在：一句话指令 → 2 分钟内获得专业晨报
- 包含：行业动态、关键公司动向、技术趋势摘要

### 案例 2：文件素材自动整理

**痛点**：内容创作者桌面和下载文件夹"爆炸"，素材管理混乱

**解决方案**：File Manager Skill

```
用户：帮我整理桌面上的图片，按月份和类型分类
龙虾：[File Manager] 扫描文件 → 按规则分类 → 移动到对应文件夹
```

## 安装建议

可以一次性安装全部 10 个基础 Skill：

```
用户：帮我安装 Google Search、Summarizer、Git、Translation 这几个 Skill
龙虾：好的，我来逐一为您安装...
```

> **建议**：先装前 3 个（Google Search + Summarizer + Git），形成最核心的办公能力闭环，再按需追加。

> [!example]- Answer Guide
> 
> #### Q1 — Top 10 Skills 筛选标准
> 
> 三个标准是**高频使用**（几乎每天都用）、**通用场景**（不限行业、大多数人适用）、**安全可信**（官方认证或社区高评分）。相比下载量，这三个标准更关注实际使用价值和安全性，避免在 1.3 万个 Skills 里迷路。
> 
> #### Q2 — Web Search 与 Summarizer 分工
> 
> Web Search Skill 负责搜索相关关键词并抓取多源信息，Summarizer Skill 负责提炼要点并输出结构化晨报；原来手动流程需要 30–60 分钟，组合后一句话指令 2 分钟内即可获得专业晨报。
> 
> #### Q3 — 最小核心装机顺序
> 
> 建议先装 **Google Search + Summarizer + Git** 这三个。Google Search 解决信息获取，Summarizer 解决内容消化，Git 解决版本回溯，三者覆盖"找信息→理解信息→管理成果"的完整办公闭环。
