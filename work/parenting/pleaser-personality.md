# 4. 讨好型人格和自卑 (People-Pleasing Personality and Inferiority)

## 讨好型人格的根源

来自对边界感的模糊，来自原生家庭：

- **隐性期待型家庭：** 父母有各种隐性期待，孩子通过满足父母需求赢得生存空间 → 得出结论：满足别人的需求才能存在
- **关系疏离型家庭：** 家里没有连接 → 孩子向外寻求连结，一旦有反对意见就被冷暴力/打骂 → 孩子误以为"没有冲突才是好的关系" → 建立的是避免冲突型关系（表面老好人，内心特别累）

**隐性期待变成显性规则，规则精简。**

当我们的边界和孩子需求有冲突时，**先站在孩子一边，帮她适应边界。先看见孩子，再重申规则。**

## 自卑的根源

自卑来自于对真实自我感知力的缺乏。

- 自卑的根源——无法感受自己的存在，自己的真实感受不断被否定
- 自卑的反面是**自爱**（比不自信更深，是一种低自我认知）
- 不自信：表层的感受
- 让自己被看见

## 核心洞察

- **轻推：** 帮助孩子扫清障碍，去争取他们想要的东西（不是直接给出办法，而是"我在"）
- 掌控感的缺失必然引起很多后续问题
- 认识目标，吵架是消耗精力的。伤害你的是**对事件的解读**。
- 学习不是为了有优越感/更高挑剔力，而是**变成一只更大更包容的容器，成为更加稳定的存在**
- 真正的惩罚：从暖变冷

## True vs Real

- 一个人的**真实感受**才是他活在这个世界上的真实拥有（在意感受，不质疑真假）
- 育儿育的是那艘船——只有船大了、稳定了，才能扬帆起航
- 自我认知是孩子的存在感
- 孩子刚存在感——困惑：我真的重要吗？
- **愿意看见你，就是爸爸妈妈对你的爱，因为你无可取代**
- 孩子从父母眼里看到自己 → 父母的镜像反馈最终内化成孩子对自我的认知
- 错的不是爸妈做了什么，而是爸妈**没有**做什么

## 成人的幸福感困境

为什么成年人把获得幸福感的主动权交出？
→ "生日你送我礼物才开心"、"你对我好才感觉到被爱" = 从来不信任自己可以主宰一段关系（父母下意识不信任孩子的结果）

## 孤独的深处

孤独的深处：是怀疑自我的价值、自我认知的问题。**有边界的爱**才是真正的爱。

培养孩子的认知：告别自卑
