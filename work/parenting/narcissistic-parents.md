# 3. 自恋型父母 (Narcissistic Parents)

## 核心原则

- 接纳感受而不是接纳行为，明确边界，给掌控感，谢谢你
- 话术：妈妈不能让…发生，妈妈现在要…了
- 奖励换成庆祝（reward → celebrate）
- 关注点：做到的东西（成功经历），接纳感受（不是做不到的）
- 用情商指导智商，用智商管理情商
- 情绪是我们身体的一部分
- 每一种情绪/感受都有一个目的（感情被忽略 → 我的判断是不准确的）
- 共情一次-短平快处理（需要妈妈的能量倾斜）
- 过度担心 → 传递信息：妈妈不信任你可以自己处理好焦虑和恐惧
- 避免权力争夺战
- 隐忍是边界模糊的表现

## 真正的共情 (Genuine Empathy)

1. 觉察自己的情绪，多层次的情绪
2. 觉察孩子的情绪，用讲述/描绘的方式帮助孩子把情绪梳理出来
3. 用"我信息"表达自己的感受，不指责、不评判，只表达
4. 深度描绘每一样情绪，建立与孩子的深度链接

## 如何做到 (Step-by-Step)

1. **我自己的情绪是什么？哪些是臆断？**
   - 臆断的潜台词：我肯定是对的

2. **孩子的情绪是什么？**
   - 用讲故事的方式梳理，建立连接

3. **我信息：**
   - 妈妈也怎么了...
   - 你想知道妈妈是怎么想的吗？→ 鼓励白色小人的部分，托举孩子

4. **边界是什么？**
   - 重申边界
   - 有温度的拒绝：我们没有这个选项
   - 标注孩子的情绪（描述状况）
   - 接纳情绪，坚持边界（妈妈不能够…）

5. **步骤：**
   - 肯定孩子的感受，通过梳理整个事情把感受尽可能罗列出来（用积极的语句）
   - 表达妈妈的感受，仅表达，不参杂评判（"我信息"）
   - 固定句式：你想不想知道妈妈是怎么想的？你是一个什么样的孩子。
   - 在跟孩子的连结很牢固时，重申边界
   - 如果孩子挑战边界，用我信息明确态度
   - 用游戏力接纳孩子感受，倾听需求，接纳任何感受

6. **孩子哭闹：** 无法消化自己的情绪
