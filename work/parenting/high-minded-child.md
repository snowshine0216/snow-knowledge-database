# 8. 高明感小孩 (High-Competence Child)

## 价值感公式

**归属感 + 胜任力 + 成就感 = 价值感**

- **胜任力：** 我能够把这件事做好的心境
- 少对孩子评头论足，少给评语 → 把注意力放在**努力**上面（避免固化思维）
- 多祝贺多祝福，培养成长心态，认识到成长空间

## 帮助搭建任务阶梯

1. **第一步：** 确认舒适区 — 地点、人、事情
2. 给孩子**非常具体**的目标，目标具体而一致
3. 培养：哪怕我说了no，也不会失去对方的自信。"妈妈相信你的人格魅力。"
4. 每一次都有成就 → 他就很愿意做这件事情

## 注意

孩子**6岁前**避免竞技活动
